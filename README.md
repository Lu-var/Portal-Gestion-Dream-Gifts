To use this application, follow these steps:

1. Clone the repository to your local machine using `git clone <repository-url>`.
2. Navigate into the project directory: `cd PortalGestionDreamGifts`.
3. Ensure that all necessary dependencies are installed by running `mvn clean install` or equivalent build command.
